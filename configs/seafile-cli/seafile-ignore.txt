# Seafile ignore patterns. Placed at the root of each synced library by
# `seafile-setup`. Keeps regenerable / junk dirs out of sync so the daemon
# isn't crawling tens of thousands of tiny files.
#
# IMPORTANT: Seafile is NOT full gitignore. A bare "name/" only matches at the
# library ROOT; nested dirs need a "*/name/" pattern too ("*" matches "/", so
# it covers any depth). Always pair both. Verified empirically.
